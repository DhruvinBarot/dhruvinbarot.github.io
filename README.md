# Dhruvin Barot — Portfolio

An interactive personal portfolio website built with vanilla HTML, CSS, and JavaScript, designed to showcase my experience, projects, technical skills, and certifications through a lightweight and visually engaging interface.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- SVG animations
- Google Fonts

## Features

- Drifting cloud background effects
- Responsive single-page layout
- Lightweight, dependency-free design

## Project Structure

- `index.html` — main content and layout
- `style.css` — styling, theme, and animations
- `script.js` — interactive behavior and motion effects
